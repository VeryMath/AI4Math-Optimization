# 测试 Prompt 集

可以直接复制以下 prompt 来测试 copt-linear-program skill。

---

## Prompt 1：基础生产问题

```
/copt-linear-program

工厂生产两种产品 A 和 B。每单位 A 利润 3 元，每单位 B 利润 5 元。
- 每单位 A 消耗 1kg 原料和 2 小时工时
- 每单位 B 消耗 2kg 原料和 1 小时工时
- 每天原料总量不超过 100kg，工时不超过 120 小时
- 产量不能为负

问：每天生产多少 A 和 B 能获得最大利润？
```

---

## Prompt 2：饮食配方问题

```
/copt-linear-program

需要配制一种混合饲料，使用原料 A 和 B。
- 原料 A 每 kg 成本 9 元，原料 B 每 kg 成本 7 元
- 每 kg A 含蛋白质 3g、维生素 1g
- 每 kg B 含蛋白质 2g、维生素 2g
- 最终饲料至少含蛋白质 100g、维生素 80g

问：各用多少 kg 的 A 和 B 成本最低？
```

---

## Prompt 3：运输问题

```
/copt-linear-program

有两个仓库 W1、W2，两个门店 S1、S2。单位运费如下：
- W1 → S1: 4 元，W1 → S2: 6 元
- W2 → S1: 5 元，W2 → S2: 3 元

库存：W1 最多 80 件，W2 最多 70 件
需求：S1 至少 60 件，S2 至少 50 件

问：如何调配使总运费最低？
```

## Prompt 9：三变量资源配置

```
/copt-linear-program

某厂生产三种产品 P1、P2、P3，单位利润分别为 50、80、40 元。
- 原料 A：P1 用 2 单位，P2 用 3 单位，P3 用 1 单位，总量 200
- 原料 B：P1 用 1 单位，P2 用 2 单位，P3 用 3 单位，总量 180
- 工时：P1 用 4 小时，P2 用 2 小时，P3 用 1 小时，总量 300 小时

问：如何安排生产使总利润最大？
```

## Prompt 10：多工厂运输

需求：M1 至少 70 件，M2 至少 60 件

问：如何调配使总运费最低？
```
